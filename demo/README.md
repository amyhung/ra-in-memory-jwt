# ra-in-memory-jwt Demo

## Prerequisites

The demo of `ra-in-memory-jwt` is expected to start in [Docker](https://www.docker.com) et [Docker-Compose](https://docs.docker.com/compose/).

## Installation

```bash
$ make install
$ make init
$ make start
```

The default user is `myFirstUser/n33dToB3+Str0ng`.
